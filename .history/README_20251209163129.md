# Contador de Páginas PDF com Envio WhatsApp

Script Python que conta páginas de arquivos PDF por pasta e envia os resultados via WhatsApp.

## Pré-requisitos

1. **Python 3.x** com PyPDF2 instalado:
   ```bash
   pip install -r requirements.txt
   ```

2. **Node.js** instalado (versão 14 ou superior)

3. **Dependências Node.js**:
   ```bash
   npm install
   ```

## Configuração

1. Edite o arquivo `config.json`:
   ```json
   {
     "whatsapp": {
       "type": "group",
       "target": "Nome do Grupo ou ID do Grupo",
       "enabled": true
     }
   }
   ```

2. Para enviar para um **grupo**:
   - `"type": "group"`
   - `"target": "Nome Exato do Grupo"` (o nome como aparece no WhatsApp)

3. Para enviar para um **contato individual**:
   - `"type": "number"`
   - `"target": "5511999999999"` (código do país + número, sem espaços ou caracteres especiais)

4. Para desabilitar o envio WhatsApp:
   - `"enabled": false`

## Primeira Execução

Na primeira vez que executar, será necessário escanear o QR Code do WhatsApp Web:

1. Execute o script Python normalmente
2. Um QR Code aparecerá no terminal
3. Abra o WhatsApp no celular
4. Vá em Configurações > Aparelhos conectados > Conectar um aparelho
5. Escaneie o QR Code exibido no terminal
6. A sessão será salva automaticamente para próximas execuções

## Uso

Execute o script normalmente:
```bash
python pagecounter-folders.py
```

O script irá:
1. Contar todas as páginas PDF em cada pasta
2. Exibir os resultados no terminal
3. Enviar automaticamente os resultados para o WhatsApp configurado

## Notas

- A primeira execução pode demorar mais devido à autenticação WhatsApp
- A sessão do WhatsApp é salva na pasta `.wwebjs_auth/` (não commitada no git)
- Se o grupo não for encontrado, o script listará os grupos disponíveis
